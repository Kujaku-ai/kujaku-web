New patterns invented

Per the updated brand authority rule, primitives not in brand/recipes.css (tabs, horizontal session strip, timeline, 2×2 tableau) were invented scene-local per variant using only brand tokens. Each invention is catalogued in NEW_PATTERNS.md for later promotion review.
